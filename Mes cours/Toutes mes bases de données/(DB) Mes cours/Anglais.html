This text was written by Oliver Grillmeyer in the journal Springer published in 1999. It has for name : ‚ÄúComputer, it is computers, their design and algorithms‚Äù. It covers hardware, software and AI, including machine learning, expert systems, and graphics. Operating systems manage resources and facilitate interaction.

Computers science is the study of the treatment of information through various computing methods in two main fields: hardware and software.

Hardware and software associate the physical and virtual parts of the computer. They are know to be the two founding field in the develpment of the computer technology. Mathematical models in the design of the virtual part of the computer allow to optimize the treatment of information within hardware.

An operating system (O.S.) provides an environement which allows the user to interact whith hardware and software. The operating system allocates ressources for user to run several programs to manage several tasks and introductions. For exemple, it manages the dynamic memory (the RAM), network communications etc.

The hardwarelanguage is the assembly language (assembleur). Writing sophisticated programs with this language is almost impossible. This is the reason why the high level langages have been developed as a link between hardware and programmers. High level languages may be considered as a compromise between the human language and the machine language.

Compilers transpose high-level instruction (such as C instructions) into the machine language. C instructions need the gcc to translate programs into the assembly language.

Today, we have compiled and interpreted high-level languages. Compiled high-level languages such as Python, Java, need interpreters to simulate the executions of instructions.
